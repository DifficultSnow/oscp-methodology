🔍 Initial Recon

Fire all scans before doing anything else. Background them with & so everything runs in parallel while you set up other windows.

Environment Variables + Directory Structure

set vars + create dirs (run in "notes" window)
export KALI=$(ip a show tun0 | grep 'inet ' | awk '{print $2}' | cut -d/ -f1)
export DC01=10.10.10.10
export DC02=10.10.10.11
export DC03=10.10.10.12
export WEB01=10.10.10.20
export LIN01=10.10.10.30
export WIN01=10.10.10.40
export DOMAIN=corp.local

mkdir -p ~/exam/{scans/{dc01,dc02,dc03,web01,lin01,win01},loot,exploits,report,logs,serve}
echo "KALI tun0: $KALI"

Initial Scan — All Targets Simultaneously

Run from "scans" tmux window. Background all with & so they run in parallel.
launch all initial scans at once
sudo nmap -sC -sV -O --open -oA ~/exam/scans/dc01/init $DC01 &
sudo nmap -sC -sV -O --open -oA ~/exam/scans/dc02/init $DC02 &
sudo nmap -sC -sV -O --open -oA ~/exam/scans/dc03/init $DC03 &
sudo nmap -sC -sV -O --open -oA ~/exam/scans/web01/init $WEB01 &
sudo nmap -sC -sV -O --open -oA ~/exam/scans/lin01/init $LIN01 &
sudo nmap -sC -sV -O --open -oA ~/exam/scans/win01/init $WIN01 &
echo "All initial scans running..."

Full Port Scan

all 65535 ports — run in parallel with initial
sudo nmap -p- --min-rate 5000 --open -oA ~/exam/scans/dc01/allports $DC01 &
sudo nmap -p- --min-rate 5000 --open -oA ~/exam/scans/web01/allports $WEB01 &
sudo nmap -p- --min-rate 5000 --open -oA ~/exam/scans/lin01/allports $LIN01 &
sudo nmap -p- --min-rate 5000 --open -oA ~/exam/scans/win01/allports $WIN01 &

Targeted Scan — After allports Finishes

extract ports + deep targeted scan
# Auto-extract discovered ports:
ports=$(grep -oP '\d+/open' ~/exam/scans/web01/allports.gnmap | cut -d/ -f1 | tr '\n' ',' | sed 's/,$//')
echo "Ports: $ports"
sudo nmap -p $ports -sC -sV -A -oA ~/exam/scans/web01/targeted $WEB01

UDP Scan

Top 25 UDP ports is enough for OSCP. Full UDP scan is too slow for exam time.
UDP top 25 — catches SNMP (161), TFTP (69), DNS (53)
sudo nmap -sU --top-ports 25 -oA ~/exam/scans/web01/udp $WEB01 &
sudo nmap -sU --top-ports 25 -oA ~/exam/scans/lin01/udp $LIN01 &
sudo nmap -sU --top-ports 25 -oA ~/exam/scans/win01/udp $WIN01 &

Autorecon — All-in-One Alternative

autorecon (more thorough, slower)
sudo autorecon $WEB01 $LIN01 $WIN01 -o ~/exam/scans/ --only-scans-dir
# or with known ports to save time:
sudo autorecon $WEB01 --ports 80,443,8080 -o ~/exam/scans/

Time Budget Guidance
