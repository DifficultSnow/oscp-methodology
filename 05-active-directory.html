queries to run immediately — in this order
critical
# In BloodHound UI → Search bar → click these built-in queries:

# 1. Find all Domain Admins
#    → Who is DA? Mark them as High Value. Find shortest path TO them.

# 2. Find Shortest Paths to Domain Admins
#    → The kill chain. Follow every edge.

# 3. Find Principals with DCSync Rights
#    → Anyone besides DA with DCSync = instant domain dump path.

# 4. Find all users with path to DA (raw Cypher):
MATCH p=shortestPath((u:User)-[*1..]->(g:Group {name:"DOMAIN ADMINS@CORP.LOCAL"})) RETURN p

# 5. Kerberoastable users with path to DA:
MATCH (u:User {hasspn:true}) RETURN u.name,u.description

# 6. AS-REP roastable users:
MATCH (u:User {dontreqpreauth:true}) RETURN u.name

# 7. All computers where current user is local admin:
MATCH (u:User {name:"<USER>@CORP.LOCAL"}),(c:Computer),p=((u)-[:AdminTo]->(c)) RETURN p

# 8. Users with GenericAll / GenericWrite / WriteDACL / ForceChangePassword:
MATCH p=(u:User)-[:GenericAll|GenericWrite|WriteDACL|ForceChangePassword]->(v) RETURN p

# 9. Computers with unconstrained delegation:
MATCH (c:Computer {unconstraineddelegation:true}) RETURN c.name

# 10. Find owned principals with path to DA (right-click node → Mark as Owned first):
MATCH p=shortestPath((o {owned:true})-[*1..]->(g:Group {name:"DOMAIN ADMINS@CORP.LOCAL"})) RETURN p
